# Installation and Usage Instructions for iPhone PWA

## Installation

1. Open Safari on your iPhone.
2. Navigate to the URL of the PWA you want to install.
3. Tap the 'Share' button (the box with an arrow pointing up).
4. Scroll down and select 'Add to Home Screen'.
5. Optionally, edit the name of the app before tapping 'Add'. This will create an icon on your home screen.

## Usage

- Once added, you can tap the app icon on your home screen to launch the PWA.
- The PWA will work offline after the initial load, depending on its service worker setup.
- You can also configure settings for notifications and updates within the app itself if supported.
